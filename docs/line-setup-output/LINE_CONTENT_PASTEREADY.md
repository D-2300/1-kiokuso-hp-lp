## Phase 1: 基本プロフィール

### 1.1 アカウント名
```
記憶荘｜仙台の店舗内装
```
※ 20文字以内。一度変更すると7日間ロックされる点に注意。

### 1.2 ステータスメッセージ（20字以内）
```
店舗の物件探しから内装まで手伝います
```

### 1.3 自己紹介（page.line.biz の「自由記述」パーツ）
パーツ名: `記憶荘とは`

```
仙台・宮城県で、店舗を始めるオーナーの「物件探し〜設計〜施工」を一社で担当しています。

内装工事会社のため物件売買では儲けない中立ポジション。「この物件、商売として回るのか」を施工側の目で見極めます。

居抜き活用・施主支給・DIY併走など、品質を落とさずに予算を圧縮する工夫も併せてご提案します。
```

### 1.4 基本情報（page.line.biz）
| 項目 | 値 |
|---|---|
| 概要（1行） | 仙台の店舗内装を、物件探しから届出まで一気通貫 |
| 業種カテゴリ | 不動産・建設 → 内装・リフォーム |
| 住所 | 宮城県仙台市（※正式な所在地を入力） |
| 電話番号 | （※代表電話） |
| 営業時間 | 平日 9:00〜18:00 |
| Webサイト | https://kiokuso.jp/ |

### 1.5 応答設定
- 応答時間内: **手動チャット＋応答メッセージ**
- チャット: **ON**
- あいさつメッセージ: **ON**
- Webhook: **OFF**

## Phase 2: あいさつメッセージ（3通・即時・分割のまま）

### メッセージ① テキスト
```
はじめまして🏠
記憶荘の 伊藤 です。

友だち追加ありがとうございます！

弊社は仙台・宮城県で店舗を始める方の
「物件探し → 設計 → 施工 → 届出」を
一社でご一緒している内装工事会社です。

内装業なので物件売買で儲ける立場ではなく、
「この物件で商売が成り立つか」を
中立の視点でお話しできます。

まずはテナント攻略ガイドをお送りしますね👇
```

### メッセージ② テキスト（テナント攻略ガイド案内）
```
📖 テナント攻略ガイド｜店舗物件・内装の読み方

下記リンクから無料でご覧いただけます👇
https://kiokuso.jp/studio/articles/mikoukai-bukken-uragawa

【記事の内容】
・「未公開物件＝お宝」は本当か
・初期段階の非公開物件にある「強気の値付け」
・店舗物件を探す前に決めるべき3つの基準
・居抜きとスケルトン、費用の違い
・内装屋から見た物件の失敗例

ご質問はこのLINEでお気軽に💬
```

### メッセージ③ テキスト（次のステップ）
```
下のメニューからアクセスできます👇

① 物件リクエスト
　「物件リクエスト」と送信 → 条件ヒアリング

② 相談予約
　「相談」と送信 → 電話 or LINE相談

③ 施工事例
　https://kiokuso.jp/studio/works

※ 営業電話はしません。ブロックも自由です。
「この物件はやめた方がいい」も正直にお伝えします。

※ すべて「友だち追加直後（即時）」送信、ディレイ0秒。3通まとめて即時配信する。

## Phase 4: キーワード自動応答（6件）

### 4.1 「物件リクエスト」
キーワード: `物件リクエスト` / `物件` / `物件情報` / `物件リスト`
```
ありがとうございます！
お探しの条件を伺ったうえで、
合うテナント物件が出た際にお声がけしています。

以下の3つを教えてください🙏

1️⃣ エリア希望
（例：国分町、仙台駅周辺、泉中央、仙台市全域など）

2️⃣ 業種と坪数イメージ
（例：カフェ15坪前後、美容室10坪、焼肉20坪など）

3️⃣ 家賃の上限目安
（例：月20万円まで、坪1万円以下）

テナント情報は仲介会社や大家さんから
お預かりした信頼の上で流通しているため、
出店意思の明確な方に限り個別にお渡ししています。
あらかじめご了承ください🙏

※ 物件まだ決まっていない段階でも大丈夫です。
「相談」と送っていただければ、条件の整理から
LINEでご一緒します。
```

## Phase 5: ステップ配信（6通 / Day 2,5,9,14,21,30）

全体期間: 30日
対象: 新規友だち全員
配信時刻: 18:00 or 20:00（下記メッセージ別）

### Day 2（追加2日後 18:00） テキスト
```
こんばんは、伊藤です🌙

昨日お送りしたテナント攻略ガイドは
ご覧いただけましたか？

ガイドの中でも特にお伝えしたいのは
「非公開物件＝お得 とは限らない」ということ。

初期段階の非公開物件はむしろ
強気の値付けのことも多く、
同じ物件が数ヶ月後に一般公開されることも
普通にあります。

大事なのは「その物件で商売が成り立つか」を
先に見ること。

候補物件があれば、そのままLINEに送ってください。
こちらで読み取って正直な感想をお返しします📩
```

### Day 5（追加5日後 20:00） テキスト
```
物件を探し始めると、内見の数だけ迷いが増えます。

でも実は、内見する前に
「自分の中で基準を決めておく」だけで、
迷う時間が大幅に減ります。

・客層は誰か（ターゲット）
・どんな形で迎えたいか（席数・滞在時間・単価）
・集客方法は何か（路面依存か、SNS依存か）

この3つが決まっていないまま物件を見ると、
どの物件も「悪くない気がする」状態になります。

📖 店舗物件を探す前にやるべきこと
https://kiokuso.jp/studio/articles/bukken-sagashi-kijun

読んで「自分の基準が曖昧かも」と感じたら、
そのままLINEで相談してください📩
```

### Day 9（追加9日後 18:00） テキスト
```
物件を見る時に、
内装屋の立場から必ず確認するポイントがあります💡

① 電気容量
飲食店なら動力（三相200V）+ 30A以上が必要。
契約後に「容量が足りません」と判明するケースも。

② 給排水・ダクト
配管・ダクトの位置が自由度を決めます。
ラーメン店・焼肉店はダクト経路が最重要。

③ 天井高・搬入経路
業務用冷蔵庫・オーブンが入るか。
エレベーター・階段幅で搬入不可になることも。

これを内見の時点で判断できると、
「買って失敗」を大幅に減らせます。

実際の物件で判断したい方は、
物件のお写真を送っていただければ
私たちが代わりに見極めますよ👀
```

### Day 14（追加14日後 20:00） テキスト
```
もし現在検討中の物件があれば、
一度ご相談いただけませんか🙏

・「この家賃で、本当に商売が回るのか」
・「内装費はいくら見込めばいいのか」
・「この立地、客は来るのか」

こうしたご質問は、
LINEで写真と情報を送っていただければ、
24時間以内にお返事します。

現地で見るのが一番早い場合は、
仙台・宮城県内であれば
物件に同行して一緒に見ます。

お気軽に「相談」と送ってください📞
```

### Day 21（追加21日後 18:00） テキスト
```

【バスケットボール Cafe & Bar CROSSOVER】
・工期：約10日間

✔️ オーナーのコレクションを内装に転用
✔️ 閉店店舗からの什器引き取りでコスト圧縮

「このくらいの予算でここまで変わるんだ」
という参考になれば嬉しいです📸

詳しい施工事例はこちら
https://kiokuso.jp/studio/works/crossover

## Phase 7: 自己テスト手順

### 7.1 友だち追加テスト
1. 別のスマホ or 別LINEアカウントで記憶荘公式を友だち追加
2. **あいさつ3通**が即時届くか確認（① 自己紹介 → ② テナント攻略ガイド案内 → ③ 次のステップ）
3. テナント攻略ガイドリンクが開けるか確認

### 7.2 キーワード応答テスト（6件）
以下を1件ずつ送信し、想定通りの返答が来るか確認：

| 送信ワード | 期待される応答 |
|---|---|
| `物件リクエスト` | エリア・坪数・家賃・業種・居抜きのヒアリング |
| `ガイド` | ガイドURL＋目次案内 |
| `相談` | 電話・LINE・現地の3択提示 |
| `料金` | 料金目安＋pricingページURL |
| `事例` | worksページURL＋代表事例 |
| `退会` | ブロック手順案内 |

### 7.3 リッチメニュータップテスト（6枠）
- 左上「物件リクエスト」→ 自動でテキスト送信
- 中上「テナント攻略ガイド」→ 自動でテキスト送信
- 右上「相談予約」→ 自動でテキスト送信
- 左下「施工事例」→ /studio/works が開く
- 中下「料金表」→ /studio/pricing が開く
- 右下「Webサイト」→ / が開く

### 7.4 ステップ配信プレビュー
- LINE Manager → ステップ配信 → 各メッセージの「プレビュー」で文面確認
- 配信時刻（18:00 / 20:00）が設定されているか
- Day 2 / 5 / 9 / 14 / 21 / 30 の6通すべて有効になっているか

### 7.5 タグ手動付与テスト
- 自分の友だちプロフィール画面でタグ5種が選択肢に出るか確認
- 試しに1つ付けてみて、リスト絞り込みが効くか確認

### 7.6 終了処理
- テスト完了後、テストアカウントは**ブロック解除のまま削除**しない（友だち数に影響）
- もしくは「友だち削除」してから本番運用へ
