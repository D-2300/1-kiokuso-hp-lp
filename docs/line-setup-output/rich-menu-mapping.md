## タイル構成（2行×3列）

| 位置 | タイル名 | アクション種別 | 値 |
|---|---|---|---|
| 1行目・左 | 物件リクエスト | テキスト送信 or LINEチャット入力 | `物件情報を送ります：\n・エリア：\n・坪数：\n・家賃上限：\n・業種：` |
| 1行目・中 | テナント攻略ガイド | URLリンク | `https://kiokuso.jp/studio/articles/mikoukai-bukken-uragawa` |

## 座標（LINE Manager 設定用）

LINE リッチメニューは `(x, y, width, height)` で領域を指定します。2500×1686 の場合：

| タイル | X | Y | 幅 | 高さ |
|---|---|---|---|---|
| 1行目・左 | 0 | 0 | 833 | 843 |
| 1行目・中 | 833 | 0 | 834 | 843 |
| 1行目・右 | 1667 | 0 | 833 | 843 |
| 2行目・左 | 0 | 843 | 833 | 843 |
| 2行目・中 | 833 | 843 | 834 | 843 |
| 2行目・右 | 1667 | 843 | 833 | 843 |

## 注意

- 「物件リクエスト」と「相談予約」は、LINE Manager でアクションを「テキスト送信」に設定する
- 他4タイルは「URL」アクションで設定
- タイトル（Default表示）は「menu」など任意
